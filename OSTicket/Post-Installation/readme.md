## Step 1️⃣: Configuring Roles
1. Go to [Admin Control Panel](http://localhost/osTicket/scp/login.php ) & click on Admin Panel in top right.
2. **Admin Panel -> Agents -> Roles**
  Roles are for group permissons. We can add a new role or edit a role to give or deny access to a certain group of people we want to assign it to.
3. To add a role, click **Add New Role** & name it something like *Supreme Admin*, then click on **Permissions** 
4. Under *tickets*, *Tasks*, & *Knowledgebase* we can check off every box. (For a *Supreme Admin* we'd want them to be able to manage everything)
